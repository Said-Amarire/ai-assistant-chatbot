== Description ==
AI Assistant ChatBot Full Control is a powerful and flexible AI chatbot plugin for WordPress.  
It allows you to add an interactive AI assistant to your website, helping you:  
- Provide instant customer support  
- Answer frequently asked questions  
- Engage visitors 24/7  

This plugin is fully editable from the WordPress Admin dashboard, making it perfect for beginners and advanced users alike.  

Enhance your website with a smart chatbot that works seamlessly with any WordPress theme. No coding required!  

You are free to use, modify, and distribute it under the GPL2 license.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access settings from the WordPress Admin dashboard to configure the chatbot.
4. The chatbot interface will appear in the footer of your website.

== Features ==
* Fully customizable chat widget
* Easy setup via WordPress Admin
* Supports translations via standard WordPress localization (.pot) files
* Loads automatically in the footer
* Compatible with all WordPress themes
* jQuery is automatically enqueued
* Optionally deletes plugin settings on uninstall
* Free to use, modify, and share
* Secure and stable for any website

== Usage ==
* The chatbot automatically loads in the footer of your site.
* Customize its behavior and appearance from the WordPress Admin panel.
* Translations are supported through standard WordPress localization files.
* jQuery is automatically loaded if not already present in your theme.

== Uninstall ==
To remove all plugin settings on uninstall, uncomment the line in `uninstall.php`:

    // delete_option('ai_assistant_chatbot_options');
